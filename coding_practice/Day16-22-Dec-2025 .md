# Artifacts Promotion

## ðŸ”¹ Jenkins Pipeline â€” Promote Artifacts
You will learn how to **promote build artifacts from dev to staging**.

pipeline {
  agent any
  environment {
    INSTITUTE_NAME = "Pathnex"
  }
  stages {
    stage('Build') {
      steps {
        sh 'mvn clean package'
      }
    }
    stage('Promote'){
      steps {
        input message: "Promote artifacts to staging?"
        sh 'echo "Promoting artifacts for $INSTITUTE_NAME"'
      }
    }
  }
}

<!-- What this pipeline does 
Runs on any Jenkins agent
Sets a global environment variable
Builds the application using Maven
Waits for manual approval
Promotes artifacts to staging (simulated with echo) -->
